AI Gesture PPT Controller

Control Presentation Slides using real-time Hand Gestures.

How it works:

  • MediaPipe: Detects and tracks hand landmarks.
  • Finger Logic: Identifies gestures from finger positions.
  • PyAutoGUI: Sends keyboard inputs for slide control.
  • NumPy + OpenCV: Creates a clean black tracking screen.

Understanding the Logic

The system detects gestures by checking whether fingers are UP or DOWN using hand landmarks.

How it works:

  • Index Finger: Next Slide
  • Thumb: Previous Slide
  • Index + Middle: Pointer Mode
  • Time Delay: Prevents multiple triggers

def fingers_up(lmList):
    tips = [4, 8, 12, 16, 20]
    fingers = []

    if lmList[tips[0]][0] > lmList[tips[0]-1][0]:
        fingers.append(1)
    else:
        fingers.append(0)

    for i in range(1, 5):
        if lmList[tips[i]][1] < lmList[tips[i]-2][1]:
            fingers.append(1)
        else:
            fingers.append(0)

    return fingers

Finger Detection Logic

Determines which fingers are UP or DOWN using hand landmark positions.

  • tips: Landmark indices of all fingertips
  • Thumb: Checked using horizontal (x-axis) position
  • Other Fingers: Checked using vertical (y-axis) position
  • Logic: Tip above joint → Finger is UP (1), else DOWN (0)
  • Output: Returns a list like [0,1,0,0,0]

while True:
    success, frame = cap.read()
    if not success:
        continue

    # BLACK SCREEN
    img = np.zeros_like(frame)

    imgRGB = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)
    results = hands.process(imgRGB)

    if results.multi_hand_landmarks:
        for handLms in results.multi_hand_landmarks:
            lmList = []

            for id, lm in enumerate(handLms.landmark):
                h, w, c = frame.shape
                cx, cy = int(lm.x * w), int(lm.y * h)
                lmList.append((cx, cy))

            fingers = fingers_up(lmList)
            current_time = time.time()

            # NEXT
            if fingers == [0,1,0,0,0] and current_time - last_action > 1:
                pyautogui.press("right")
                pyautogui.press("down")
                last_action = current_time

            # PREVIOUS
            if fingers == [1,0,0,0,0] and current_time - last_action > 1:
                pyautogui.press("left")
                pyautogui.press("up")
                last_action = current_time

            # POINTER
            if fingers == [0,1,1,0,0]:
                x, y = lmList[8]
                cv2.circle(img, (x, y), 15, (0, 0, 255), cv2.FILLED)

            # DRAW ONLY LANDMARKS ON BLACK SCREEN
            mpDraw.draw_landmarks(img, handLms, mpHands.HAND_CONNECTIONS)

    cv2.imshow("PPT Controller", img)

    if cv2.waitKey(1) & 0xFF == 27:
        break
cap.release()
cv2.destroyAllWindows()

Main Execution Loop

Continuously captures frames, detects gestures, and controls slides in real-time.

  • Frame Capture: Reads live video from webcam
  • Black Canvas: Displays only landmarks for clean UI
  • Hand Processing: Converts frame to RGB & detects hand
  • Landmark Mapping: Converts normalized points to pixel values
  • Gesture Detection: Uses finger logic to identify actions
  • Next Slide: Index finger → Right/Down key press
  • Previous Slide: Thumb → Left/Up key press
  • Pointer Mode: Index + Middle → Draw pointer on screen
  • Delay Logic: Prevents multiple rapid triggers
  • Exit: Press ESC to close application
